# 03. 실전 프로젝트: 고객 응대 자동화

> 들어오는 문의를 분류하고, 답변을 생성하고, 담당자에게 라우팅하는 시스템

---

## 프로젝트 개요

**목표**: 고객 문의 처리 자동화

**처리 흐름**:
1. 고객 문의 수신 (이메일/폼)
2. AI가 문의 유형 분류
3. FAQ는 자동 답변
4. 복잡한 건은 담당자에게 전달 (요약 포함)

**가치**: 1차 응대 80% 자동화, 응답 시간 수일 → 수분

---

## 에이전트팀 구조

```
[고객 문의 수신]
       ↓
[Classifier Agent]
  문의 유형 분류:
  - FAQ (자주 묻는 질문)
  - 불만/환불
  - 기술 문제
  - 기타
       ↓
  ┌────────────────────────┐
  ↓                        ↓
[FAQ Agent]          [Escalation Agent]
자동 답변 생성        담당자에게 라우팅
FAQ DB 검색           요약 + 우선순위 판단
  ↓                        ↓
[Quality Checker]    [Notification Agent]
답변 품질 검토        담당자에게 Slack 알림
       ↓
[최종 답변 전송]
```

## Step 1: FAQ 데이터베이스 구성

**faq.json** 파일 생성:

```json
{
  "faqs": [
    {
      "category": "배송",
      "question": "배송 기간은 얼마나 걸리나요?",
      "answer": "일반 배송은 3~5 영업일, 빠른 배송은 1~2 영업일 소요됩니다.",
      "keywords": ["배송", "도착", "언제", "기간"]
    },
    {
      "category": "환불",
      "question": "환불 신청은 어떻게 하나요?",
      "answer": "구매 후 7일 이내, 미사용 제품에 한해 환불 가능합니다. 고객센터(1234-5678)로 연락 주세요.",
      "keywords": ["환불", "반품", "취소"]
    },
    {
      "category": "계정",
      "question": "비밀번호를 잊어버렸어요",
      "answer": "로그인 페이지의 '비밀번호 찾기'를 클릭하고 이메일을 입력하시면 재설정 링크를 보내드립니다.",
      "keywords": ["비밀번호", "로그인", "계정", "찾기"]
    }
  ]
}
```

## Step 2: CLAUDE.md 설정

**CLAUDE.md**

```markdown
# 고객 응대 자동화 시스템

## 목적
고객 문의를 분류하고 적절히 처리한다.

## 응대 원칙
- 항상 친절하고 공감적으로
- 명확하고 간결하게
- 해결책을 먼저, 이유는 나중에
- 고객 이름을 호칭에 사용

## FAQ 데이터
faq.json 파일을 참조한다.

## 에스컬레이션 기준
다음 경우 담당자에게 전달:
- 환불 금액 10만원 초과
- 법적 문제 언급
- 3번 이상 반복 문의
- 심한 불만 표현
```

---

## Step 3: 핵심 Skill 작성

**.claude/commands/handle-inquiry.md**

```markdown
고객 문의를 처리해줘.

문의 내용은 내가 아래에 붙여넣을게.

처리 단계:

**1. 분류**
문의 유형을 판단해줘:
- [FAQ] 자주 묻는 질문으로 자동 처리 가능
- [COMPLAINT] 불만/환불 처리 필요
- [TECHNICAL] 기술 문제
- [ESCALATE] 담당자 개입 필요

**2. FAQ 처리 (해당 시)**
faq.json에서 관련 답변을 찾아서
고객 이름을 포함한 맞춤형 답변을 생성해줘.

**3. 에스컬레이션 (해당 시)**
담당자에게 전달할 요약을 만들어줘:
- 문의 요약 (2~3줄)
- 우선순위: 높음/중간/낮음
- 권장 처리 방법

**4. 결과 저장**
processed/{날짜}/{고객ID}.json 형식으로 저장해줘.
```

## Step 4: 테스트 실행

```bash
claude
```

```
/handle-inquiry

문의 내용:
고객명: 김철수
문의: 지난주에 주문한 상품이 아직도 안 왔어요.
      주문번호는 ORD-20260101-123입니다.
      빨리 처리해주세요.
```

---

## Step 5: 배치 처리

여러 문의를 한 번에 처리:

```
inbox/ 폴더에 있는 오늘 수신된 고객 문의 파일을 모두 처리해줘.
FAQ 건은 자동 답변 생성, 에스컬레이션 건은 목록으로 정리해줘.
처리 결과 요약을 마지막에 보여줘.
```

## 성과 측정

처리 후 이런 통계를 요청할 수 있습니다:

```
이번 달 처리된 고객 문의 통계를 만들어줘.
- 총 건수
- 자동 처리 건수 vs 에스컬레이션 건수
- 유형별 분류
- 평균 응답 시간 (파일 생성 시각 기준)
```
