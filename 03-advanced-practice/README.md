# 03 고급 실습 — 에이전트팀으로 서비스 만들기

> 여러 AI가 협력하는 시스템을 설계하고, 실제 서비스를 처음부터 끝까지 구축한다

---

## 이 과정은 누구를 위한 것인가?

- AI로 실제 동작하는 서비스를 만들고 싶은 분
- 에이전트팀 구조를 설계하고 싶은 분
- 개발 경험이 있거나, 기초 실습을 완료한 분
- 반복 업무를 완전히 자동화하는 시스템을 구축하고 싶은 분

---

## 이 과정을 마치면

- [ ] 에이전트팀의 역할과 구조를 설계할 수 있다
- [ ] Orchestrator-Worker 패턴으로 복잡한 작업을 자동화한다
- [ ] 실제로 동작하는 서비스를 처음부터 끝까지 만든다
- [ ] 만든 서비스를 배포하고 운영할 수 있다
