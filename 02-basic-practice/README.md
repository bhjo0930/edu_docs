# 02 기초 실습 — 도구를 설치하고 써보기

> Claude Code를 설치하고, MCP로 외부 서비스를 연결하고, Skill로 반복 작업을 자동화한다

---

## 이 과정은 누구를 위한 것인가?

- AI 도구를 직접 설치해서 써보고 싶은 분
- 반복 업무를 자동화하고 싶은 분
- 개발 경험은 없지만 도전해보고 싶은 분
- **터미널(검은 창)을 처음 써보는 분도 따라할 수 있습니다**

---

## 이 과정을 마치면

- [ ] Claude Code를 설치하고 기본 사용법을 익힌다
- [ ] MCP를 연결해서 Slack, Notion 같은 도구와 Claude를 연동한다
- [ ] 자주 쓰는 작업을 Skill로 만들어 한 줄로 실행한다
- [ ] 내 업무에 맞는 간단한 자동화를 만든다
