# AI 활용 교육

> 수준별로 선택하는 AI 활용 실전 교육

---

## 과정 선택

### 어떤 과정이 맞나요?

```
"AI가 뭔지 알고 싶고,          → 01 일반
 어떻게 쓰이는지 보고 싶어"

"직접 써보고 싶고,             → 02 기초 실습
 도구를 설치해서 자동화해보고 싶어"

"실제 서비스를 AI로 만들고      → 03 고급 실습
 에이전트 시스템을 구축하고 싶어"
```

## 과정 개요

| 과정 | 대상 | 내용 | 전제 조건 |
|------|------|------|-----------|
| [01 일반](./01-general/README.md) | 누구나 | AI 트렌드, 국내외 사례, 웹 IDE 체험 | 없음 |
| [02 기초 실습](./02-basic-practice/README.md) | 도구를 써보고 싶은 분 | Claude Code 설치, MCP 연결, Skill 활용 | 컴퓨터 기초 사용 |
| [03 고급 실습](./03-advanced-practice/README.md) | 서비스 구축 목표 | 에이전트팀 설계, 실제 서비스 개발 | 기초 실습 완료 또는 개발 경험 |

## 각 과정 상세

### 01 일반 — AI를 이해하고 체험하기

```
1. AI 트렌드 — 지금 AI는 어디까지 왔나
2. 바이브 코딩 — 코딩 없이 만드는 새로운 방법
3. AI 비서 — 목표를 주면 알아서 처리하는 AI
4. 국내외 사례 — 실제로 이렇게 쓰고 있다
5. 웹 IDE 실습 — Lovable, AI Studio, Replit으로 직접 만들기
```

### 02 기초 실습 — 도구를 설치하고 써보기

```
1. 환경 설정 — Claude Code 설치
2. Claude Code 기초 — 대화로 파일 만들기
3. MCP 연결 — 외부 서비스 연결하기
4. Skill 활용 — 반복 작업 자동화
```

### 03 고급 실습 — 서비스 만들기

```
1. 에이전트팀 설계 — 여러 AI가 협력하는 구조
2. 실전 프로젝트 — 실제 서비스를 처음부터 끝까지
3. 배포와 운영 — 만든 것을 세상에 내보내기
```
