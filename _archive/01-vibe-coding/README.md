# 모듈 01: 바이브 코딩 (Vibe Coding)

> "코딩을 모르는 사람도, AI와 함께라면 프로덕트를 만들 수 있다"

---

## 모듈 개요

**바이브 코딩(Vibe Coding)** 은 AI와 자연어 대화를 통해 소프트웨어를 만드는 방식입니다.
전통적인 코딩 지식보다 **무엇을 만들지 명확히 표현하는 능력**이 핵심입니다.

### 핵심 철학

- 완벽한 코드를 처음부터 쓰려 하지 않는다
- AI에게 방향을 주고, 결과를 검토하고, 반복한다
- "어떻게(How)"보다 "무엇을(What)"에 집중한다

---

## 학습 목표

이 모듈을 마치면:
- [ ] 바이브 코딩이 무엇인지 설명할 수 있다
- [ ] Claude를 브라우저에서 바로 사용할 수 있다
- [ ] 자연어로 간단한 웹 페이지를 만들 수 있다
- [ ] 효과적인 지시 방법(프롬프트 패턴)을 3가지 이상 활용한다
- [ ] 내 업무에 맞는 미니 결과물 1개를 완성한다
