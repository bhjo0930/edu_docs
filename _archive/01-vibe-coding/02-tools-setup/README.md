# 02. 도구 설정

AI와 대화를 시작하기 위한 준비입니다. 설치할 것이 거의 없습니다.

---

## 방법 1: 웹 브라우저로 바로 시작 (가장 쉬움)

**설치 없이 지금 바로 시작할 수 있습니다.**

1. [claude.ai](https://claude.ai) 접속
2. 계정이 없다면 무료 가입
3. 채팅창에 원하는 것을 말하면 됩니다

이걸로 충분합니다. 바이브 코딩의 대부분은 이 방법으로 할 수 있습니다.

---

## 방법 2: Claude Code 데스크톱 앱 (권장)

파일을 직접 만들고 저장하는 작업을 할 때 편합니다.

**Mac / Windows 앱 설치:**
1. [claude.ai/download](https://claude.ai/download) 에서 앱 다운로드
2. 일반 앱처럼 설치
3. 로그인 후 바로 사용

> **데스크톱 앱의 장점**: 만든 파일이 내 컴퓨터에 저장됩니다.
> 웹 버전은 채팅 창 안에서만 결과를 볼 수 있어요.

---

## 방법 3: Claude Code CLI (심화 — 선택)

> ⚠️ 이 방법은 터미널(검은 창)을 사용합니다.
> 비개발자라면 방법 1~2로도 충분합니다. 나중에 익숙해지면 도전해보세요.

터미널을 통해 Claude Code를 사용하면 더 강력한 자동화가 가능합니다.

설치 방법은 [공식 문서](https://docs.anthropic.com/claude-code)를 참고하세요.

---

## 어떤 방법을 선택해야 할까?

| 상황 | 추천 방법 |
|------|-----------|
| 처음 시작, 바로 해보고 싶다 | 방법 1 (웹 브라우저) |
| 파일을 만들어 저장하고 싶다 | 방법 2 (데스크톱 앱) |
| 개발 업무도 하거나 자동화를 깊이 파고 싶다 | 방법 3 (CLI) |

---

## Claude 계정 만들기

1. [claude.ai](https://claude.ai) 접속
2. **Sign Up** 클릭
3. 이메일 또는 Google 계정으로 가입
4. 이메일 인증 완료

**무료 플랜으로도** 대부분의 실습이 가능합니다.
메시지 전송 횟수에 제한이 있으니, 중요한 작업에 집중해서 사용하세요.

---

## 첫 번째 테스트

계정이 생겼다면 바로 해보세요:

[claude.ai](https://claude.ai) 채팅창에 입력:

```
안녕! 나를 소개하는 간단한 웹 페이지 HTML을 만들어줘.
이름: [본인 이름]
직업: [본인 직업]
취미: [본인 취미]
```

결과로 HTML 코드가 나오면, 그것을 복사해서 `.html` 파일로 저장하고
브라우저로 열어보세요. 나만의 소개 페이지가 생겼습니다!
