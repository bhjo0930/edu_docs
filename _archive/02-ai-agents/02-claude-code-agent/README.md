# 02. Claude Code — AI 비서 사용하기

> 파일을 직접 열고, 읽고, 만들고, 수정하는 AI 비서

---

## Claude Code가 일반 Claude와 다른 점

**일반 Claude (claude.ai 웹):**
- 채팅창에서 대화
- 내 컴퓨터 파일에 접근 불가
- 내가 내용을 붙여넣어야 함

**Claude Code:**
- 내 컴퓨터 파일을 직접 읽고 수정
- 여러 파일을 한 번에 처리
- 결과물을 파일로 직접 저장

---

## 시작하기

### 방법 1: 데스크톱 앱 사용 (비개발자 권장)

1. [claude.ai/download](https://claude.ai/download) 에서 앱 설치
2. 앱 실행 후 로그인
3. 작업할 폴더를 앱에서 열기
4. 대화창에 원하는 작업 입력

### 방법 2: 웹 브라우저 사용

파일을 다루지 않는 작업이라면 [claude.ai](https://claude.ai) 웹에서 충분합니다.

> 터미널(CLI) 방법은 [심화 섹션](../06-claude-code-automation/README.md)을 참고하세요.

---

## 실제 업무 활용 사례

### 사례 1: 폴더 안 문서 한 번에 정리

**상황**: 팀원들이 올린 주간 업무 보고서 파일 20개를 하나로 요약해야 할 때

**Claude Code에게 지시:**
```
이 폴더에 있는 주간보고서 파일들을 모두 읽어서
팀원별로 이번 주 완료한 일, 다음 주 예정 일, 이슈를 정리해줘.
결과를 "주간요약_[날짜].md" 파일로 저장해줘.
```

Claude Code가 하는 일:
1. 폴더 안 파일 목록 확인
2. 각 파일 하나씩 읽기
3. 팀원별 정보 추출
4. 요약 파일 생성

---

### 사례 2: 고객 데이터 정리

**상황**: CSV 파일에 고객 정보가 있는데 형식이 제각각일 때

```
customer_data.csv 파일을 열어서:
1. 전화번호 형식을 모두 "010-0000-0000" 형식으로 통일해줘
2. 이메일이 없는 고객을 별도 목록으로 만들어줘
3. 정리된 파일을 "customer_clean.csv"로 저장해줘
```

---

### 사례 3: 반복 문서 자동 생성

**상황**: 거래처별로 같은 형식의 견적서를 보내야 할 때

```
"거래처목록.xlsx"를 읽어서
각 거래처마다 맞춤형 견적서를 만들어줘.
양식은 "견적서_양식.docx"를 참고하고
각 거래처 폴더에 저장해줘.
```

---

### 사례 4: 웹에서 정보 수집 후 정리

```
우리 업계 주요 뉴스를 오늘 날짜 기준으로 검색해서
주제별로 정리한 브리핑 문서를 만들어줘.
```

---

## 효과적인 지시 방법

### 파일을 다룰 때는 파일명을 명시하세요

```
나쁜 예:  "데이터 정리해줘"
좋은 예:  "sales_2025.csv 파일을 정리해줘"
```

### 원하는 결과 형식을 알려주세요

```
나쁜 예:  "요약해줘"
좋은 예:  "요약해서 '팀별 요약.xlsx' 파일로 저장해줘"
```

### 모르는 것은 물어보라고 하세요

```
"파일을 분석하다가 이해가 안 되는 부분이 있으면
 진행하기 전에 먼저 물어봐줘"
```

---

## 주의사항

**중요한 파일은 작업 전에 복사해두세요.**
Claude Code가 파일을 수정할 때 실수할 수 있습니다.
원본 폴더를 복사해서 백업을 만든 뒤 작업하는 습관을 들이세요.

**결과물은 반드시 확인하세요.**
AI가 만든 결과물은 항상 사람이 검토해야 합니다.
특히 숫자(금액, 날짜, 수량)는 꼭 확인하세요.
