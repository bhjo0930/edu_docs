# 03. 여러 비서 활용하기 (Subagent)

> "큰 프로젝트는 여러 명에게 나눠 맡기는 것이 빠르다"

---

## 1. 하나의 AI 비서로 충분하지 않을 때

혼자 처리하기에 너무 크거나 복잡한 작업이 있습니다.

**예시:**
- 50개의 고객 인터뷰 파일을 분석해서 인사이트 도출
- 회사 전체 계약서를 검토하고 만료일 목록 만들기
- 100개 상품의 경쟁사 가격을 조사하고 비교표 만들기

이런 경우, AI 비서가 스스로 **여러 명의 보조 비서(Subagent)** 를 만들어 작업을 나눠 처리합니다.

---

## 2. 비서팀 운영 방식

```
[나]
  ↓ "고객 인터뷰 50개를 분석해서 주요 불만 사항을 정리해줘"
[메인 비서]
  ↓ 작업 분배
  ├── [보조 비서 1] 인터뷰 파일 1~17번 분석
  ├── [보조 비서 2] 인터뷰 파일 18~34번 분석
  └── [보조 비서 3] 인터뷰 파일 35~50번 분석
  ↓ 결과 취합
  → "주요 불만 사항 TOP 10 + 원문 근거" 보고서 완성
```

일반 비서 1명이 순서대로 50개를 처리하는 것보다 **3~5배 빠릅니다.**

---

## 3. 일상 업무 적용 사례

### 사례 1: 대량 문서 처리

**상황**: 올해 제출된 제안서 30개를 검토해야 함

```
이 폴더의 제안서 파일 30개를 검토해줘.
각 제안서에서:
- 제안 금액
- 납기일
- 기술적 위험 요소
를 추출하고, 전체를 비교한 표를 만들어줘.
빠르게 처리할 수 있도록 병렬로 처리해도 돼.
```

→ Subagent가 여러 파일을 동시에 처리해서 빠르게 완료

---

### 사례 2: 다각도 분석

**상황**: 신규 서비스 런칭 전 종합 분석 필요

```
우리 신규 서비스 기획서를 4가지 관점으로 동시에 분석해줘:
1. 고객 입장에서 본 장단점
2. 운영/비용 관점에서 실현 가능성
3. 경쟁사 대비 차별점
4. 잠재적 리스크
각 분석 결과를 종합해서 최종 의견도 줘.
```

→ 4가지 분석을 동시에 진행하고 결과를 통합

---

### 사례 3: 업무 분리

**상황**: 보고서 작성 + 데이터 수집을 동시에

```
다음 두 가지를 동시에 해줘:
1. 지난달 판매 데이터를 분석해서 그래프용 데이터 준비
2. 이번 달 목표 달성 현황 글을 작성

두 결과를 합쳐서 경영진 보고서 초안을 만들어줘.
```

---

## 4. 내가 직접 지시하는 방법

비서팀을 활용하려면 특별한 설정이 필요한 게 아닙니다.
Claude Code에서 이렇게 말하면 됩니다:

```
[작업 설명]
파일이 많다면 병렬로 처리해줘.
```

또는:

```
이 작업을 여러 부분으로 나눠서 동시에 처리하면 더 빠를 것 같아.
알아서 분배해서 처리해줘.
```

Claude Code가 판단해서 Subagent를 활용할지 결정합니다.

---

## 5. 주의할 점

**정확도보다 속도가 필요할 때 활용하세요.**
여러 비서가 동시에 처리하면 빠르지만, 중요한 작업은 결과를 꼭 검토해야 합니다.

**작업 내용을 명확하게 주세요.**
"알아서 해줘"보다 구체적인 지시가 더 좋은 결과를 냅니다.

---

## 실습

아래 시나리오 중 내 업무에 해당하는 것이 있나요?

- 많은 파일을 한 번에 처리해야 하는 작업
- 같은 내용을 여러 관점에서 분석해야 하는 작업
- 동시에 진행할 수 있는 독립적인 여러 작업

Claude에게 물어보세요:
```
나는 [직무]야.
[작업 설명]을 해야 하는데
비서 여러 명을 활용하면 더 효율적으로 할 수 있어?
어떻게 지시하면 좋을지 알려줘.
```
