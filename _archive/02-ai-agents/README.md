# 모듈 02: AI 비서 활용

> "목표를 주면 알아서 처리하는 AI, 어떻게 활용할까?"

---

## 모듈 개요

**AI 비서(Agent)** 는 지시를 받으면 스스로 계획을 세우고, 필요한 작업을 차례로 처리한 뒤 결과를 가져옵니다.

매번 내가 지시하지 않아도 됩니다. 목표를 주면 알아서 합니다.

---

## 바이브 코딩 vs AI 비서

| | 바이브 코딩 | AI 비서 |
|--|------------|---------|
| 방식 | 내가 묻고, AI가 답 | 목표를 주면 AI가 자율 실행 |
| 작업 규모 | 한 번의 요청 | 여러 단계로 이어지는 작업 |
| 사람의 역할 | 매 단계 확인 | 목표 설정 + 최종 결과 검토 |
| 적합한 상황 | 빠른 결과물 제작 | 반복·복잡한 업무 자동화 |

---

## 학습 목표

이 모듈을 마치면:
- [ ] AI 비서가 무엇인지, 어떻게 활용하는지 설명할 수 있다
- [ ] Claude Code로 파일 여러 개를 한 번에 처리할 수 있다
- [ ] 하위 비서(Subagent)를 활용해 큰 작업을 나눠서 처리할 수 있다
- [ ] 내 업무에 맞는 자동화 시나리오를 직접 설계할 수 있다
