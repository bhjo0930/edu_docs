# 01. AI 비서란?

> "지시를 받으면 스스로 계획하고, 알아서 처리하고, 결과를 가져오는 AI"

---

## 1. 일반 AI vs AI 비서의 차이

우리가 아는 ChatGPT나 Claude는 **질문-답변** 방식입니다.
내가 물으면 답하고, 또 물으면 또 답합니다.

**AI 비서(Agent)** 는 다릅니다.
목표를 주면 **스스로 알아서 처리**합니다.

---

### 비유로 이해하기

**일반 AI** = 매우 똑똑한 사람에게 전화로 물어보기

```
나: "우리 팀 지난달 매출 보고서 어떻게 만들어?"
AI: "엑셀에서 이렇게 하면 돼요: ..."
나: (직접 해야 함)
```

**AI 비서** = 비서에게 업무를 맡기기

```
나: "지난달 매출 보고서 만들어줘"
AI 비서: 파일 찾기 → 데이터 정리 → 보고서 작성 → "완성했습니다"
나: (결과만 받으면 됨)
```

---

## 2. AI 비서가 할 수 있는 일

### 문서/파일 작업
- 폴더 안의 파일들을 모두 읽고 요약
- 여러 문서를 합쳐서 하나의 보고서로 정리
- 문서에서 특정 정보 찾아서 정리

### 정보 수집
- 웹에서 관련 자료 검색하고 정리
- 여러 페이지의 내용을 비교 분석

### 자동화
- 반복적인 파일 처리 작업
- 데이터 형식 변환

### 코드/개발 (개발자 활용)
- 코드 분석 및 수정
- 버그 찾기 및 수정

---

## 3. 일반 AI와 비교해보기

| | 일반 AI (Claude.ai) | AI 비서 (Claude Code) |
|--|-----|------|
| 사용 방법 | 채팅으로 물어보기 | 목표를 주면 알아서 실행 |
| 파일 접근 | 내가 붙여넣어야 함 | 내 컴퓨터 파일 직접 접근 |
| 작업 단위 | 한 번의 질문 | 여러 단계로 이어지는 작업 |
| 적합한 용도 | 글쓰기, 아이디어, 설명 | 파일 처리, 자동화, 반복 작업 |

---

## 4. 실제 활용 사례

### 사례 1: 회의록 정리

```
[폴더에 회의록 파일 10개가 있을 때]

나: "이 폴더의 모든 회의록에서 '다음 할 일'로 정리된 내용만
    뽑아서 담당자별로 정리해줘"

AI 비서: 파일 하나씩 읽기 → 할 일 추출 → 담당자별 정리 → 결과 파일 생성
```

### 사례 2: 경쟁사 분석

```
나: "경쟁사 A, B, C의 홈페이지를 방문해서
    각 회사의 주요 제품과 가격을 비교 표로 만들어줘"

AI 비서: 각 사이트 방문 → 정보 수집 → 비교표 작성
```

### 사례 3: 이메일 초안 대량 작성

```
[고객 목록 파일이 있을 때]

나: "이 고객 목록을 보고, 각 고객의 업종에 맞게
    맞춤형 제안 이메일 초안을 각각 만들어줘"

AI 비서: 목록 읽기 → 업종 파악 → 맞춤 이메일 작성 → 파일로 저장
```

---

## 5. AI 비서를 사용할 때 알아야 할 것

### 결과는 항상 검토하세요

AI 비서는 매우 유능하지만 실수를 할 수 있습니다.
중요한 작업의 결과는 반드시 사람이 확인해야 합니다.

### 명확하게 지시할수록 좋습니다

```
모호한 지시: "보고서 만들어줘"

명확한 지시: "sales_data.xlsx 파일을 읽어서
              월별 매출 합계와 전월 대비 증감률을
              표로 정리한 보고서를 만들어줘"
```

### 되돌릴 수 없는 작업은 미리 백업

파일 삭제나 대량 수정 작업 전에는 원본을 복사해두세요.

## 실습

Claude.ai ([claude.ai](https://claude.ai))에서 아래를 입력해보세요:

```
나는 마케팅 팀에서 일해. 매달 경쟁사 3곳의 SNS 게시물을 분석해야 해.
AI 비서를 활용하면 이 작업을 어떻게 자동화할 수 있을지 단계별로 설명해줘.
```

AI가 제안하는 방법을 읽으면서 "AI 비서가 어떻게 일하는지" 감을 잡아보세요.
