01
바이브 코딩
주요 용어
AI 도구를 쓸 때 자주 나오는 핵심 용어를 정리합니다
1부

꼭 알아야 할 핵심 용어 (1)

LLM (대형 언어 모델)
ChatGPT·Claude·Gemini 같은 AI의 핵심 엔진. 방대한 텍스트를 학습해 언어를 이해하고 생성한다.
프롬프트 (Prompt)
AI에게 입력하는 지시문. "어떻게 말하느냐"에 따라 결과 품질이 크게 달라진다.
컨텍스트 (Context)
AI가 한 번에 기억할 수 있는 대화·문서의 분량. 길수록 더 많은 정보를 참조해 답한다.
토큰 (Token)
AI가 텍스트를 처리하는 최소 단위. 영어 단어 1개 ≈ 1토큰, 한국어는 글자·형태소 단위.
할루시네이션 (Hallucination)
AI가 사실이 아닌 내용을 자신 있게 말하는 현상. 수치·날짜·출처는 반드시 직접 확인 필요.
바이브 코딩 (Vibe Coding)
코드를 직접 쓰지 않고, 자연어로 '원하는 것'만 말해서 AI가 코드를 만들게 하는 방식.
1부

꼭 알아야 할 핵심 용어 (2)

AI 에이전트 (Agent)
목표를 주면 스스로 계획·실행·수정하는 AI. 단순 답변이 아니라 업무를 자율적으로 처리.
MCP (모델 컨텍스트 프로토콜)
AI가 외부 도구(파일, DB, API 등)와 연결하는 표준 인터페이스. Claude Code 등에서 활용.
RAG (검색 증강 생성)
AI가 답변 전에 외부 문서를 검색해 참조하는 방식. 최신 정보·사내 자료를 반영할 때 사용.
파인튜닝 (Fine-tuning)
범용 AI 모델을 특정 도메인·스타일에 맞게 추가 학습시키는 것. 기업 전용 AI 만들 때 사용.
MVP (최소 기능 제품)
핵심 기능만 담은 가장 작은 버전의 제품. 바이브 코딩으로 빠르게 MVP를 만드는 것이 목표.
풀스택 (Full Stack)
화면(프론트엔드) + 서버/DB(백엔드)를 모두 포함한 완전한 서비스. AI 도구가 이를 자동 구현.
1부

꼭 알아야 할 핵심 용어 (3)

서비스는 화면(프론트엔드) → 처리(백엔드) → 저장(DB) 3층으로 이루어집니다.
이 구조를 알면 AI에게 훨씬 정확하게 요청할 수 있습니다.

프론트엔드 (Frontend)
사용자가 직접 보는 화면. 버튼·입력창·목록·대시보드 등 "보여주는 곳". "버튼을 크게 해줘"는 프론트엔드 요청.
백엔드 (Backend)
버튼을 눌렀을 때 실제로 일어나는 처리. 로그인 확인·주문 처리·권한 검사 등 "처리하는 곳". 사용자 눈에 보이지 않음.
데이터베이스 (DB)
정보를 저장하는 곳. 사용자 정보·게시글·주문 내역 등 "저장하는 곳". 서비스를 껐다 켜도 데이터가 남는 이유.
API
프론트엔드와 백엔드가 대화하는 통로. "상품 목록 주세요" → API 호출 → 백엔드가 응답. 화면과 내부 처리 사이의 연결선.
컴포넌트 (Component)
프론트엔드를 이루는 작은 부품. 버튼·검색창·카드·메뉴바 등. AI에게 부품 단위로 요청하면 더 정확한 결과를 얻음.
CRUD
DB의 기본 동작 4가지. Create(생성) · Read(조회) · Update(수정) · Delete(삭제). 게시판 = 글쓰기·보기·수정·삭제.
1부

프롬프트 잘 쓰는 법

나쁜 프롬프트 ✗

"보고서 만들어줘"

"앱 만들어줘"

"고쳐줘"

→ AI가 무엇을 원하는지 추측해야 함
→ 결과가 기대와 다를 확률 높음

좋은 프롬프트 ✓

"대상(20대 직장인), 형식(카드뉴스), 포함 내용(핵심 지표 3개)을 담은 마케팅 성과 보고서 작성"

→ 역할 + 배경 + 원하는 결과 + 형식
→ 레퍼런스 이미지가 있으면 첨부

역할 지정
"마케팅 전문가처럼"
→
배경 설명
"우리 서비스는..."
→
결과 명시
"~형식으로 출력"
→
제약 조건
"300자 이내로"

2부

Google AI Studio로 시작하기

무료
Google AI Studio
Gemini 모델을 기반으로 프롬프트 실험, 웹앱·UI 생성, API 키 발급까지 한곳에서 처리. 디자인 퀄리티가 뛰어나 빠른 MVP 제작에 최적.
aistudio.google.com
Step 1
aistudio.google.com 접속 → Google 계정으로 로그인
Step 2
'Create new prompt' → 채팅 또는 'Build' 탭에서 앱 생성 시작
Step 3
원하는 화면/기능을 자연어로 설명 → 코드·UI 자동 생성
실습
"간단한 할 일 목록(Todo) 앱을 깔끔한 디자인으로 만들어줘" 입력해보기
2부

Lovable로 풀스택 앱 만들기

유/무료
Lovable
프론트엔드 + 백엔드 + DB를 자연어 한 줄로 구현. 출시 1개월 만에 월 매출 1억 달러 달성. 실제 서비스 가능한 수준의 결과물을 가장 빠르게 만드는 도구.
lovable.dev
할 수 있는 것

• 로그인/회원가입 포함 웹앱
• 데이터 저장·조회 기능
• 대시보드·관리자 화면
• 결제 연동 MVP

실습 예시 프롬프트

"팀원들이 매일 업무 일지를 작성하고 공유할 수 있는 간단한 사내 업무 일지 앱을 만들어줘. 로그인 기능 포함."

AI 기반 IDE

3부

Cursor — AI 코드 에디터

유/무료
Cursor
VS Code 기반 AI 코드 에디터. 코드베이스 전체를 이해하고 자연어로 코드 작성·수정·리뷰를 수행. Fortune 500 기업 절반 이상이 사용. OpenAI·Anthropic 등 최신 모델 선택 가능.
cursor.com
핵심 기능 1
Tab — 스마트 자동완성
다음 코드 블록을 예측해 Tab 한 번으로 완성. 단순 자동완성이 아닌 맥락 기반 예측.
핵심 기능 2
Agent — 자율 코딩
아이디어를 입력하면 에이전트가 독립적으로 파일을 생성·수정·실행. 병렬 처리 가능.
핵심 기능 3
BugBot — PR 자동 리뷰
GitHub 연동으로 Pull Request 자동 검토. 논리 버그·보안 이슈 감지 및 개선 제안.
추천 대상 & 실습
코딩 경험이 있는 분
VS Code에 익숙하다면 바로 사용 가능.
실습: 기존 HTML 파일 열고 Cmd+K → "버튼을 파란색으로 바꿔줘"
3부

Antigravity IDE 사용하기

AI IDE
Antigravity
코드베이스 전체를 이해하는 AI 에디터. 기존 코드를 읽고 맥락을 파악해 수정·추가를 자동화. 비전공자도 기존 프로젝트를 유지보수할 수 있는 수준을 목표로 설계.
antigravity.dev
핵심 기능 1
코드베이스 이해
프로젝트 전체 파일을 분석해 "이 함수가 어디서 사용되나요?" 같은 질문에 즉시 답변
핵심 기능 2
인라인 AI 수정
코드 위에서 바로 자연어로 "이 부분을 더 빠르게 바꿔줘" 입력 → 즉시 수정 제안
추천 대상
기술 담당자 / IT 팀
코딩 경험이 약간 있거나, 기존 코드를 수정·유지보수해야 하는 분께 최적
시작 방법
설치 → 폴더 열기 → 질문
VS Code 기반으로 기존 환경에 익숙하다면 바로 사용 가능. 실습: 기존 HTML 파일 열어서 "버튼 색상 파란색으로 바꿔줘"

AI 기반 CLI
터미널 도구
명령줄에서 AI와 직접 대화하며 코드·작업을 처리하는 전문가용 도구
4부

Gemini CLI

무료
Gemini CLI
Google의 Gemini 모델을 터미널에서 바로 사용. 무료 티어 제공, Google 계정만 있으면 시작 가능. 코드 작성·파일 수정·웹 검색을 명령줄에서 처리.
github.com/google-gemini/gemini-cli
설치
npm install -g @google/gemini-cli
시작
gemini 입력 → Google 계정 로그인
실습
프로젝트 폴더에서 gemini "이 폴더에 있는 파일들을 설명해줘"
4부

Claude Code

유료
Claude Code
Anthropic의 Claude 모델 기반 터미널 에이전트. 코드베이스 전체를 이해하고 파일 읽기·수정·실행을 자율적으로 처리. 복잡한 다단계 작업 자동화에 탁월.
claude.ai/code
설치
npm install -g @anthropic-ai/claude-code
시작
claude 입력 → Anthropic 계정 로그인 (API 키 필요)
실습
프로젝트 폴더에서 claude "README 파일을 한국어로 작성해줘"
4부

OpenAI Codex CLI

유료
Codex CLI
OpenAI의 GPT-4o 기반 터미널 에이전트. 자연어 지시만으로 코드 작성·테스트·수정을 자동 실행. 보안 샌드박스 모드로 안전하게 코드를 실행할 수 있는 것이 특징.
github.com/openai/codex
설치
npm install -g @openai/codex
시작
codex 입력 → OpenAI API 키 설정
실습
codex "현재 디렉토리의 Python 스크립트를 분석해줘"

도구 비교 & 선택 가이드

비개발자 / 처음 시작
AI Studio → Lovable
설치 없이 브라우저에서 바로 시작. 디자인된 UI를 빠르게 보면서 감 잡기에 최적.
코딩 경험 약간 있음
Antigravity IDE
기존 프로젝트 열어서 AI와 함께 수정·확장. VS Code에 익숙하다면 진입 장벽 낮음.
무료로 CLI 체험
Gemini CLI
Google 계정만으로 무료 사용 가능. 터미널이 처음이어도 시작하기 가장 쉬운 CLI 도구.
전문가 / 자동화 구축
Claude Code / Codex
복잡한 워크플로우, 다단계 자동화, 대규모 코드베이스 작업에 적합. API 비용 발생.
