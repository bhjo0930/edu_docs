# 01. AI 트렌드 — 지금 AI는 어디까지 왔나

---

## 1. 2025~2026 AI 변화의 핵심

### 과거: "AI가 보조 도구"
```
2022년 이전: AI = 특정 기능 자동화 (번역, 이미지 인식 등)
2023년:      ChatGPT 등장 → 자연어로 대화하는 AI 대중화
2024년:      AI가 코드를 쓰고, 문서를 만들고, 이미지를 생성
```

### 현재: "AI가 스스로 일한다"
```
2025~2026년: AI Agent 시대
- AI가 목표를 받고 스스로 계획하고 실행
- 여러 AI가 협력해서 복잡한 프로젝트 수행
- 사람은 방향만 주고 결과를 검토
```

## 3. AI Agent: 가장 중요한 변화

### Agent란 무엇인가

기존 AI는 "대화 상대"였습니다.
새로운 AI는 "업무를 처리하는 직원"입니다.

```
기존 AI:
  나: "보고서 어떻게 쓰면 돼?"
  AI: "보고서는 이렇게 씁니다: ..."
  나: (직접 써야 함)

AI Agent:
  나: "지난달 판매 데이터로 보고서 만들어줘"
  AI: 파일 열기 → 분석 → 보고서 작성 → "완성했습니다"
  나: (검토만 하면 됨)
```

### 왜 지금 주목받나

- **실용성**: 실제 업무 자동화가 가능해짐
- **접근성**: 코딩 없이도 강력한 자동화 구현
- **비용**: 인건비 대비 압도적 효율
- **속도**: 24시간 쉬지 않고 처리

## 4. 바이브 코딩: 개발 방식의 혁명

**바이브 코딩** = AI와 대화만으로 소프트웨어를 만드는 방식

2025년 Andrej Karpathy(전 OpenAI 연구 디렉터)가 명명한 개념.

```
전통적 개발:        바이브 코딩:
아이디어            아이디어
   ↓                   ↓
설계 (수일)         "이런 걸 만들어줘" (수분)
   ↓                   ↓
코딩 (수주)         AI가 코드 생성
   ↓                   ↓
테스트 (수일)       결과 확인 & 수정
   ↓                   ↓
배포                배포
```

**핵심**: 개발자가 아닌 사람도 소프트웨어를 만들 수 있게 됨

## 5. 산업별 AI 도입 현황

### 이미 AI가 바꾸고 있는 분야

| 분야 | 변화 |
|------|------|
| **소프트웨어 개발** | AI가 코드 50~80% 작성, 개발자는 검토·설계 |
| **마케팅** | 콘텐츠 생성, A/B 테스트, 개인화 자동화 |
| **고객 서비스** | 1차 문의 AI 처리, 복잡한 건만 사람이 대응 |
| **의료** | 진단 보조, 의료 기록 분석, 연구 가속 |
| **법률** | 계약서 검토, 판례 분석, 문서 초안 |
| **금융** | 사기 탐지, 리스크 분석, 투자 조언 보조 |
| **교육** | 맞춤형 학습 경로, 즉각적 피드백 |

## 6. AI 시대에 필요한 역량

### 줄어드는 역량
- 단순 정보 검색
- 반복적 문서 작성
- 기초적 코딩 (CRUD)
- 정형화된 데이터 처리

### 늘어나는 역량
- **AI 협업 능력**: AI에게 잘 지시하는 능력
- **검토와 판단**: AI 결과물을 평가하는 능력
- **창의적 기획**: AI가 못하는 "왜"와 "무엇을"
- **도메인 전문성**: AI와 결합할 때 가치 폭발

## 핵심 요약

1. AI는 "대화"에서 "행동"으로 진화 중
2. 바이브 코딩으로 비개발자도 소프트웨어 제작 가능
