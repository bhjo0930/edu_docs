# 05. 웹 IDE 실습 — 설치 없이 브라우저로 만들기

> 오늘 배운 AI 개념을 직접 체험합니다. 설치할 것 없이 브라우저만 있으면 됩니다.

---

## 웹 IDE란?

**웹 IDE** = 브라우저에서 바로 쓰는 개발 환경

설치 불필요, 로그인만 하면 바로 시작.
AI가 내장되어 있어서 대화만으로 만들 수 있습니다.

---

## 도구 비교

| 도구 | 특징 | 적합한 용도 | 가격 |
|------|------|-------------|------|
| **Lovable** | 자연어로 풀스택 앱 생성 | 웹앱, 대시보드 | 무료 플랜 있음 |
| **Bolt.new** | 즉시 동작하는 웹앱 생성 | 프로토타입 | 무료 플랜 있음 |
| **Google AI Studio** | Gemini API 실험 | 프롬프트 테스트 | 무료 |
| **Replit** | 코드 실행 환경 + AI | 스크립트, 자동화 | 무료 플랜 있음 |
| **v0 (Vercel)** | UI 컴포넌트 생성 | 웹 UI 디자인 | 무료 플랜 있음 |

## 실습 1: Lovable로 웹앱 만들기 (권장)

**Lovable**은 자연어 설명만으로 완전한 웹 앱을 만들어줍니다.

### 시작하기

1. [lovable.dev](https://lovable.dev) 접속
2. Google 계정으로 로그인
3. "New Project" 클릭
4. 채팅창에 원하는 것을 설명

### 실습 과제

**과제 A — 개인 소개 페이지**

```
나를 소개하는 포트폴리오 페이지를 만들어줘.
- 이름: [본인 이름]
- 직업: [본인 직업]
- 소개글 섹션
- 연락처 섹션
- 깔끔하고 전문적인 디자인
```

**과제 B — 업무 도구**

```
팀 주간 업무 현황을 입력하는 웹 폼을 만들어줘.
입력 항목:
- 담당자 이름
- 이번 주 완료한 일 (텍스트)
- 다음 주 계획 (텍스트)
- 이슈/블로커 (텍스트, 선택)
제출하면 목록에 쌓이고, 전체를 볼 수 있게 해줘.
```

**과제 C — 이벤트 랜딩 페이지**

```
우리 회사 [행사명] 이벤트 랜딩 페이지를 만들어줘.
- 행사 소개
- 일시: [날짜]
- 장소: [장소]
- 신청 버튼 (클릭 시 신청 완료 메시지)
- 회사 브랜드 컬러: [색상]
```

### Lovable 사용 팁

**수정하고 싶을 때:**
```
헤더 배경색을 진한 파란색으로 바꿔줘
버튼을 더 크게 만들어줘
모바일에서도 잘 보이게 수정해줘
```

**안 되는 게 있을 때:**
```
지금 만든 것에서 [문제 설명]이 안 돼.
어떻게 고칠 수 있어?
```

## 실습 2: Google AI Studio로 프롬프트 실험

**Google AI Studio**는 Gemini 모델을 직접 테스트하는 환경입니다.
다양한 프롬프트를 실험하고 최적의 지시 방법을 찾는 데 유용합니다.

### 시작하기

1. [aistudio.google.com](https://aistudio.google.com) 접속
2. Google 계정으로 로그인
3. "Create new prompt" 클릭

### 실습 과제

**시스템 프롬프트 실험**

왼쪽 "System instructions" 박스에 입력:

```
당신은 [회사명]의 고객 서비스 담당자입니다.
친절하고 전문적인 톤으로 답변합니다.
모르는 내용은 솔직하게 모른다고 말합니다.
```

오른쪽에서 고객 질문을 입력하고 AI 응답 품질을 확인해보세요.

**온도(Temperature) 조절 실험**

같은 질문을 온도 0.1 (일관적), 1.0 (창의적)으로 각각 테스트해보세요.
어떤 차이가 있나요?

## 실습 3: Replit으로 자동화 스크립트 만들기

**Replit**은 브라우저에서 실제 코드를 실행할 수 있는 환경입니다.
AI와 대화하며 간단한 자동화를 만들어볼 수 있습니다.

### 시작하기

1. [replit.com](https://replit.com) 접속
2. 계정 만들기 (GitHub 로그인 가능)
3. "Create Repl" → Python 선택
4. 우측 AI 채팅창 활용

### 실습 과제

AI 채팅창에 입력:

```
텍스트 파일을 읽어서 단어 빈도를 분석하고
가장 많이 나온 단어 10개를 출력하는 Python 스크립트를 만들어줘.
```

코드가 생성되면 Run 버튼으로 실행해보세요.

## 실습 4: v0 (Vercel) 으로 UI 만들기

**v0**는 설명만으로 웹 UI 컴포넌트를 생성합니다.

1. [v0.dev](https://v0.dev) 접속
2. GitHub 계정으로 로그인
3. 설명 입력

### 실습 과제

```
월별 매출을 보여주는 대시보드 UI를 만들어줘.
- 상단: 총 매출, 전월 대비 증감, 목표 달성률 카드 3개
- 하단: 월별 막대 그래프
- 다크 모드 디자인
```
