### [신규 1] 즉시 실습: CLAUDE.md 작성해보기
- **위치:** 1부 Paperclip 슬라이드(slide 7) 바로 뒤 (새 slide 8)
- **섹션:** s1
- **슬라이드 타입:** 일반 슬라이드 (흰 배경)
- **배지:** `1부 실습`
- **제목:** `즉시 실습: CLAUDE.md <span class="accent">직접 작성</span>`
- **내용 구성:**
  - 상단: 빈 칸 채우기 템플릿 (code-block 스타일)
    ```
    # 나의 프로젝트

    ## 프로젝트 목적
    [내가 만들려는 것을 한 줄로]

    ## 기술 스택
    [사용할 도구: Lovable / Claude Code / MCP 등]

    ## 금지 사항
    - [하지 말아야 할 것 1가지]
    ```
  - 하단: workflow 3단계
    1. 위 템플릿을 채워 `.claude/CLAUDE.md` 파일로 저장
    2. Claude Code 실행 → `/clear` 입력
    3. 첫 번째 질문 입력 → 컨텍스트가 자동 로드되는지 확인
  - quote-box: `"CLAUDE.md 한 장이 100번의 반복 설명을 대체합니다"`

### [신규 2] 즉시 실습: Supabase MCP 연결하기
- **위치:** 2부 Supabase MCP 슬라이드(slide 11, 삽입 후 12) 바로 뒤
- **섹션:** s2
- **슬라이드 타입:** 일반 슬라이드
- **배지:** `2부 실습`
- **제목:** `즉시 실습: Supabase MCP <span class="accent">연결 따라하기</span>`
- **내용 구성:**
  - code-block: `.mcp.json` 복사용 스니펫
    ```json
    {
      "mcpServers": {
        "supabase": {
          "command": "npx",
          "args": ["-y", "@supabase/mcp-server-supabase@latest",
                   "--supabase-url", "YOUR_PROJECT_URL",
                   "--supabase-api-key", "YOUR_ANON_KEY"]
        }
      }
    }
    ```
  - workflow 3단계:
    1. Supabase 프로젝트 URL + anon key 복사 (Settings → API)
    2. 프로젝트 루트에 `.mcp.json` 저장 (위 내용 붙여넣기)
    3. Claude Code 재시작 → `"테이블 목록 보여줘"` 입력
  - warn-box: `⚠ API 키는 .gitignore에 추가하거나 환경변수로 분리`

## simpleToc 변경 사항

삽입 후 slide 인덱스 전체 재조정 필요. 추가되는 TOC 항목:

```javascript
{ num: '실습', text: 'CLAUDE.md 작성 실습', slide: 8 },       // 1부 끝
{ num: '실습', text: 'Supabase MCP 실습', slide: 14 },        // 2부 끝
{ num: '실습', text: 'Agent Team 설계 실습', slide: 23 },     // 4부 끝
{ num: '실습', text: '기획 워크시트', slide: 26 },            // 5부
{ num: '실습', text: '30분 구현 타임박스', slide: 28 },       // 5부
{ num: '실습', text: '발표 스크립트 템플릿', slide: 30 },     // 5부
```

*(정확한 인덱스는 구현 시 확정)*

## 구현 시 주의사항

1. **인덱스 충돌 방지:** 슬라이드 삽입 시 simpleToc의 모든 slide 값을 재계산
2. **섹션 divider 건드리지 않기:** 기존 섹션 구조(s1~s5) 유지
3. **기존 슬라이드 내용 수정 최소화:** 3부 Skills 실습 슬라이드만 예시 코드 1개 추가
4. **CSS 재사용:** 신규 슬라이드는 기존 CSS 클래스만 활용 (code-block, workflow, strategy-list, compare-grid, warn-box, stat-grid, quote-box)
